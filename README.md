# csec464-project

 Forensics Final Project - CSEC464

## Running

* Edit the `docker-compose.yml` environment variables to your settings
* `docker-compose up --build -d`

### Env Vars

Server

* `("REDIS_HOST", "redis")`
* `("REDIS_PORT", "6379")`
* `("REDIS_PASSWORD", "")`
* `("COMPANY_NAME", "CSEC464")`

Redis

* `REDIS_PASSWORD`
